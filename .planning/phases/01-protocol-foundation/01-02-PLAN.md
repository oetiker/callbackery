- operationId: logout
- tags: [Authentication]

- operationId: getCurrentSession

Create plugin and navigation endpoints - the core of "backend configures frontend":

**spec/paths/plugins.yaml:**

`GET /plugins`:
- operationId: listPlugins
- tags: [Plugins]
- security: bearerAuth required
- responses:
  - 200: ResponseEnvelope with array of Plugin summaries (id, type, name, title)
  - 401: Unauthorized

`GET /plugins/{pluginId}`:
- operationId: getPlugin
- tags: [Plugins]
- parameters: pluginId (path, required)
- security: bearerAuth required
- responses:
  - 200: ResponseEnvelope with Plugin
  - 401: Unauthorized
  - 404: NotFound (PLUGIN_NOT_FOUND)

`GET /plugins/{pluginId}/config`:
- operationId: getPluginConfig
- tags: [Plugins]
- description: "Returns UI configuration for the plugin - defines what the frontend should render"
- parameters: pluginId (path), parentId (query, optional - for hierarchical context)
- security: bearerAuth required
- responses:
  - 200: ResponseEnvelope with PluginConfig (type-specific: form fields, table columns, etc.)
  - 401: Unauthorized
  - 404: NotFound

`GET /plugins/{pluginId}/data`:
- operationId: getPluginData
- tags: [Plugins]
- description: "Returns data for the plugin (table rows, form initial values, etc.)"
- parameters:
  - pluginId (path)
  - offset (query, integer, default 0)
  - limit (query, integer, default 50)
  - sort (query, string, optional - column:asc or column:desc)
  - filter (query, object, optional - column filters)
  - parentId (query, optional - hierarchical context)
- security: bearerAuth required
- responses:
  - 200: ResponseEnvelope with data array + PaginationMeta
  - 401: Unauthorized
  - 404: NotFound

`POST /plugins/{pluginId}/data`:
- operationId: processPluginData
- tags: [Plugins]
- description: "Process data submission (form save, action execution)"
- parameters: pluginId (path)
- requestBody: action (string), data (object)
- security: bearerAuth required
- responses:
  - 200: ResponseEnvelope with result
  - 201: ResponseEnvelope with created resource
  - 401: Unauthorized
  - 404: NotFound
  - 422: ValidationError (field-level errors)

**spec/paths/navigation.yaml:**

`GET /navigation`:
- operationId: getNavigation
- tags: [Navigation]
- description: "Returns the application navigation structure (menus, tabs)"
- security: bearerAuth required
- responses:
  - 200: ResponseEnvelope with Navigation (array of NavigationItem, recursive)
  - 401: Unauthorized

`GET /navigation/{itemId}/children`:
- operationId: getNavigationChildren
- tags: [Navigation]
- description: "Returns children of a navigation item (for lazy loading)"
- parameters: itemId (path)
- security: bearerAuth required
- responses:
  - 200: ResponseEnvelope with array of NavigationItem
  - 401: Unauthorized
  - 404: NotFound
